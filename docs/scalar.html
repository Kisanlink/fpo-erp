🏷️ Tax System Overview

✅ Automatic Tax Calculation

Taxes are automatically calculated during sales transactions based on inventory batch configuration.

🇮🇳 GST Compliant

Proper CGST + SGST calculation with nearest paisa rounding for Indian GST compliance.

🔧 Custom Tax Support

Unlimited additional taxes via tax IDs - perfect for cess, fees, and special charges.

🆓 Tax Exemption

Full support for tax-free products and special exemption cases.

Create Inventory Batch with Tax Configuration

POST /api/v1/batches { "warehouse_id": "WH_BANGALORE_001", "product_id": "PROD_RICE_BASMATI", "cost_price": 85.50, "expiry_date": "2025-12-31", "quantity": 1000, // 🆕 NEW: Tax Configuration "cgst_rate": 2.5, // 2.5% CGST "sgst_rate": 2.5, // 2.5% SGST (Total GST = 5%) "custom_tax_ids": [ // Optional: Additional custom taxes "TAX_CESS_ENV_001", "TAX_MANDI_FEE_001" ], "is_tax_exempt": false // Whether this batch is tax-exempt }

Sale Item Response (with Automatic Tax Calculation)

{ "id": "SITM_12345678", "sale_id": "SALE_12345678", "batch_id": "BTCH_12345678", "quantity": 10, "selling_price": 100.00, "line_total": 1000.00, // 🆕 NEW: Calculated tax amounts "cgst_amount": 25.00, // 2.5% of 1000 "sgst_amount": 25.00, // 2.5% of 1000 "custom_tax_amount": 10.00, // Custom taxes "total_tax_amount": 60.00 // Total tax }

📋 Key API Endpoints

Inventory Management

GET /api/v1/batches - List all inventory batches
POST /api/v1/batches - Create inventory batch with tax config NEW
GET /api/v1/batches/{id} - Get specific batch by ID
GET /api/v1/batches/expiring - Get batches expiring soon

Sales Management

GET /api/v1/sales - List all sales
POST /api/v1/sales - Create sale (auto tax calculation) ENHANCED
GET /api/v1/sales/{id} - Get specific sale by ID

Tax Management

GET /api/v1/taxes - List all taxes
POST /api/v1/taxes - Create custom tax
POST /api/v1/taxes/calculate - Calculate tax for items

Product & Warehouse Management

GET /api/v1/products - List all products
POST /api/v1/products - Create new product
GET /api/v1/warehouses - List all warehouses
POST /api/v1/warehouses - Create new warehouse

🔒 Authentication

All API endpoints require JWT authentication. Include your token in the Authorization header:

Authorization: Bearer YOUR_JWT_TOKEN_HERE

Example with curl:

curl -X POST http://localhost:3000/api/v1/batches \ -H "Authorization: Bearer YOUR_JWT_TOKEN" \ -H "Content-Type: application/json" \ -d '{ "warehouse_id": "WH_001", "product_id": "PROD_001", "cost_price": 100.00, "expiry_date": "2025-12-31", "quantity": 500, "cgst_rate": 2.5, "sgst_rate": 2.5, "is_tax_exempt": false }'
